# ClawsCrew Skill

## Purpose
Foundation skill scaffold for ClawsCrew. Add abilities, tooling notes, and usage instructions here.

## Quick Start
- Update `config/skill.config.example.json` with real values.
- Use `bin/mission-control` for local CLI entrypoint (stub).

## Structure
- `bin/mission-control` — CLI stub entrypoint
- `config/` — configuration templates
- `SKILL.md` — this document
